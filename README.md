# Student Performance Analysis Using Machine Learning

## Project Overview

This project explores the factors that influence student academic performance (CGPA) using Data Science and Machine Learning techniques.

The objective is to identify which academic, behavioral, and lifestyle factors have the strongest impact on student success.

---

## Objective

To analyze student-related attributes and determine how factors such as attendance, study hours, sleep patterns, social media usage, stress levels, and learning habits affect academic performance.

---

## Dataset

The dataset used in this project is a synthetic dataset containing 200 student records.

Features include:

* Attendance
* Study Hours
* Sleep Duration
* Stress Level
* Social Media Usage
* Time Management
* Motivation Level
* Exercise Frequency
* Class Participation
* Exam Preparation Habits

Target Variable:

* CGPA

---

## Methodology

### Data Preprocessing

* Data Cleaning
* Categorical Encoding
* Feature Transformation

### Exploratory Data Analysis

* Correlation Analysis
* Distribution Analysis
* Feature Relationship Visualization

### Machine Learning

* Random Forest Regressor
* Feature Importance Analysis
* SHAP Explainability

---

## Key Findings

### Positive Factors

* Higher attendance showed the strongest positive relationship with CGPA.
* Students who studied more hours generally achieved better academic performance.
* Adequate sleep was associated with higher CGPA.

### Negative Factors

* Excessive social media usage negatively affected academic performance.
* Higher stress levels were linked to lower CGPA.
